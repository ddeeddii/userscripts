# Userscripts

A collection of small userscripts (featuring currently none) and userstyles that I've made to enhance my browsing experience (hence they are very opinionated), primarily debloating websites.

# Usage
For userstyles, use an extension like Stylus and download them via userstyles.world or manually by copying the CSS code into a new style.

For userscripts, this section will be updated once I have something to share.

**All of them are intended to be used alongside an ad blocker** and are primarily designed with the desktop versions in mind.
# Userstyles

## Fandom Debloater
Majorly overhauls the increasingly annoying and bloated Fandom wikis, removing most of the things people don't care about.
 
More specific information can be found on the userstyles.world page, linked [here](https://userstyles.world/style/3252/fandom-debloater). Features a toggleable search bar.


## Twitter Debloater
Removes some of the elements on the left side such as stuff related to Premium and Grok.
